# note book

A minimalist note book with the possibility for embedded components, featuring an AI companion.

## Overview
"note book" is designed to provide a clean, distraction-free baseline experience for drafting, sketching, and organizing thoughts. It serves as a blank canvas where users can seamlessly embed rich media, code snippets, Pantone color swatches, and hand-drawn sketches.

## The Baseline Experience
The initial experience is intentionally sparse and focused. It strips away complex navigation and cluttered toolbars, letting the content speak for itself. 

## The AI Companion
To enhance the drafting experience without breaking the minimalist aesthetic, "note book" includes an AI companion. The companion adapts to the user's needs through various specialized personas:

1. **The Professional**: Formal, precise, and business-oriented.
2. **Work Forward**: Action-driven, concise, and focused on productivity.
3. **Calm and Collected**: Relaxed, mindful, and supportive.
4. **Children**: Playful, encouraging, and easy to understand.
5. **The Architect**: Structural, logical, and systems-oriented.
6. **The Crew Chief**: Direct, practical, and solution-focused.
7. **The Student**: Curious, inquisitive, and eager to explore.
8. **The Professor**: Academic, deeply knowledgeable, and instructive.

These personas allow the AI companion to adjust its tone and functionality based on the user's current mindset and task.

## Hosting & Deployment
This project is built using React and Vite, with a Node/Express backend for API integrations.

### Local Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
npm start
```
